概要

本資料は、2026年7月7日に実施された第2回キックオフミーティングの議事メモである。NDA契約、API連携インターフェース仕様書、全体スケジュール、体制・役割、リリース目標、連携仕様の提供時期など、前回課題と協議事項の進捗が整理されている。

主要事項

  • NDA契約は京成電鉄側で進行中だが、提出前提であれば資料共有は可能との認識。
  • API連携I/F仕様書は日立製作所が現行版を来週目処、正式版を7月末目標で提供する想定。
  • 12月結合は厳しく、リリース目標を1月～2月、最遅3月中旬で再調整する流れ。
  • 京成・NSS・日立の3者調整がスケジュールの重要要素になっている。
