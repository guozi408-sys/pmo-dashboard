概要

本資料は、NSS社内用のSharePointフォルダ管理ルールを説明する資料である。ルート直下はPMが管理し、定期的な整理整頓、バージョン番号や更新日の付与、最新ファイルの識別ルールを定めている。

主要事項

  • SharePointフォルダはNSS社内用ファイル共有フォルダとして管理する。
  • ルートフォルダ直下はPMが管理し、定期的に整理整頓する。
  • 最新ファイルが分かるように、ファイル名にバージョン番号または更新日を付与する。

判断候補

  • ファイル名にバージョン番号または更新日を付与する運用とする。
    candidate / SLIDE-001

確認事項候補

  • プロジェクト成果物、議事録、社内資料の正式な配置ルールと命名規則。
    NSS PM / SLIDE-001

リスク候補

  • 命名規則やフォルダ管理が徹底されない場合、最新版判断や資料追跡が難しくなる可能性。
    low / SLIDE-001

アクション候補

  • SharePointフォルダの命名・配置ルールをPMO運用ルールに反映する。
    candidate / NSS PMO / SLIDE-001
