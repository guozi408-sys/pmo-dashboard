概要

本資料は、京成電鉄向けWeChatミニプログラム導入提案書である。中国人インバウンド向けに自社WeChatミニプログラムを構築し、Skyliner乗車券購入、顔認証、チケットレス化、マーケティング活用、将来的なインバウンドプラットフォーム展開を提案している。

主要事項

  • 他社ミニプログラム依存から自社ミニプログラムへ移行することで、ブランド、データ蓄積、販促、利用コスト削減を狙う。
  • Ph.1は引換QR購入・表示、Ph.1.1は顔写真撮影・アップロード、Ph.2は座席指定・乗車QR発券を想定。
  • トップページ、チケット購入、顔写真撮影、決済、チケット詳細、乗車QRなどの画面遷移案が含まれる。
  • 第二・第三フェーズでは観光地チケット、飲食店クーポン、沿線観光・地域連携への展開を想定。

確認事項候補

  • 見積前提となるPh.1/Ph.1.1/Ph.2の正式スコープ。
